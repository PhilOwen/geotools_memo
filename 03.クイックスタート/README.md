GeoTools公式チュートリアルの最初、
Quickstartをやってみる。
少し変更して、Scalaで。

EsriのShapefile形式のファイルを開いて、
ウィンドウに表示してみる。

シェープは、国土地理院からダウンロードしたもので、
パンやズームできるようにした。  
地球地図日本というもの。原則フリーで使える。

起動がちょっと遅い。
また、終了時に警告が出ているが、スルー。
